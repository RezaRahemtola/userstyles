# userstyles

Dark themes and clean site redesigns for popular websites, published to [userstyles.world](https://userstyles.world/user/RezaRahemtola?page=1&sort=mostinstalls) and
[userstyles.org](https://userstyles.org/user-profile/1068424), with the source mirrored here.

## Install

1. Install **Stylus**
   ([Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) ·
   [Firefox](https://addons.mozilla.org/firefox/addon/styl-us/) ·
   [Opera](https://addons.opera.com/extensions/details/stylus/))
2. Install a theme, either:
   - **Directly from GitHub:** click the
     [![Install](https://img.shields.io/badge/Install-238b8b)](#themes)
     button next to a theme below. Stylus detects the `.user.css` and opens an install dialog, and updates are pulled from this repo automatically.
   - **From the directories:** search the theme name on
     [userstyles.world](https://userstyles.world) (Stylus) or
     [userstyles.org](https://userstyles.org) (Stylish or Stylus).

## Maintainer automation

This repo ships Claude Code skills, agents, and slash commands used to build and
maintain these themes: `/theme-discover` (find + auto-build new candidates),
`/theme-build`, `/theme-review`, and `/theme-patrol` (re-check published themes for
breakage).\
Browser work is driven by `@playwright/cli` (headless bundled Chromium),
one isolated session per agent. One-time setup:

```sh
npm install
npx playwright-cli install-browser chromium
```
