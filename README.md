# Userstyles

Dark themes and clean site redesigns for popular websites, for the
[Stylus](https://github.com/openstyles/stylus) and Stylish userstyle managers.

Each theme is published to [userstyles.world](https://userstyles.world) and
[userstyles.org](https://userstyles.org), with the source mirrored here.

## Install

1. Install a userstyle manager — **Stylus**
   ([Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) ·
   [Firefox](https://addons.mozilla.org/firefox/addon/styl-us/)), recommended, or **Stylish**.
2. Install a theme:
   - **Stylus + GitHub (auto-updating):** open the theme's `.user.css` file.
   - **From the directories:** search the theme name on
     [userstyles.world](https://userstyles.world) (Stylus) or
     [userstyles.org](https://userstyles.org) (Stylish or Stylus).
